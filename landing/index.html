Get started

One command.
You're done.

No configuration files. No docker-compose. Just run this and open your browser.

Single command install
docker run -d --name cinestack -p 80:80 \
  -v cinestack-data:/var/lib/postgresql/data \
  -v /mnt/media:/mnt/media \
  ghcr.io/homesoftco/cinestack:latest
Requires Docker + Jellyfin + Prowlarr + qBittorrent. Full install guide →
